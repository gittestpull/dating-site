# GoldRush - Premium Private Matching Service

최상위 1%를 위한 고품격 프라이빗 데이팅 서비스, GoldRush입니다.

## 🚀 시작하기

### 필수 조건
- Node.js 18.x 이상
- npm 또는 yarn

### 설치 및 실행
```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

서버가 실행되면 [http://localhost:3000](http://localhost:3000)에서 프로젝트를 확인할 수 있습니다.

## ✨ 주요 기능
- **프리미엄 메인 페이지**: 세련된 다크 모드와 골드 테마의 히어로 섹션
- **AI 기반 매칭 UI**: Tinder 스타일의 매력적인 카드 스와이프 인터페이스
- **프로필 갤러리**: 다양한 멤버들의 프로필을 한눈에 확인
- **프라이빗 메시지**: 매칭된 인연과의 대화방 UI
- **고급 멤버십 관리**: 신원 인증 및 VVIP 멤버십 프로필 UI
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 완벽 대응

## 🛠 기술 스택
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Icons**: Lucide React
- **Language**: TypeScript

## 📂 프로젝트 구조
- `src/app`: 페이지 라우팅 및 레이아웃
- `src/components`: 재사용 가능한 UI 컴포넌트
- `src/lib`: 상수, 유틸리티 함수 및 타입 정의
- `public/profiles`: AI 생성 프로필 이미지 샘플
